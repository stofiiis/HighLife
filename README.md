# HighLife (NeoForge)

Author: **stofiiis**

Minecraft version: **1.21.11**  
NeoForge version: **21.11.38-beta**

## Features
- Cannabis seeds drop from grass/fern blocks.
- Cannabis crop with 4 growth stages.
- Crop harvesting yields buds and seeds.
- Drying rack block with block entity processing (`cannabis_bud` -> `dried_cannabis_bud`).
- Consumables: `joint`, `pipe`, `bong`, `weed_brownie`.
- Custom effects: `Relaxed`, `Cottonmouth`, `Fog`.
- Hidden tolerance mechanic with passive decay.
- Overdose side effect chance when chain-using smokeables.
- Custom internet-sourced textures with attribution in [TEXTURE_CREDITS.md](TEXTURE_CREDITS.md).

## Development
- Build: `gradlew.bat build`
- Run client: `gradlew.bat runClient`
- Run dedicated server: `gradlew.bat runServer`
