# HighLife (NeoForge)

Author: **stofiiis**

Minecraft version: **1.21.11**  
NeoForge version: **21.11.38-beta**

## Features
- Mystic herb seeds drop from grass/fern blocks.
- Mystic herb crop with 4 growth stages.
- Crop harvesting yields buds and seeds.
- Drying rack block with block entity processing (`mystic_herb_bundle` -> `dried_mystic_herb`).
- Consumables: `herb_roll`, `infusion_wand`, `alchemy_flask`, `herb_cookie`.
- Custom effects: `Serenity`, `Thirst`, `Haze`.
- Hidden tolerance mechanic with passive decay.
- Overuse side effect chance when chain-using infusion tools.
- Custom internet-sourced textures with attribution in [TEXTURE_CREDITS.md](TEXTURE_CREDITS.md).

## Development
- Build: `gradlew.bat build`
- Run client: `gradlew.bat runClient`
- Run dedicated server: `gradlew.bat runServer`
